# modelosUML

Fuentes `.puml` de todos los diagramas del proyecto. La jerarquía interna espeja la de [`RUP/`](/RUP/):

```
modelosUML/RUP/00-requisitos/...  ← diagramas del requisitado (heredados del SDR)
modelosUML/RUP/01-analisis/...    ← diagramas de colaboración por caso de uso
modelosUML/RUP/02-diseño/...      ← diagramas de secuencia por caso de uso
```

Cada `.puml` tiene su `.svg` renderizado en la ruta análoga bajo [`/images/`](/images/). Las READMEs de cada artefacto bajo `RUP/` enlazan a ambos.
