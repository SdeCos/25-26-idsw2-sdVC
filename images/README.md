# images

Imágenes del proyecto: `.svg` renderizados de los diagramas PlantUML y `.png` de los prototipos. La jerarquía interna espeja la de [`RUP/`](/RUP/):

```
images/RUP/00-requisitos/...  ← SVGs/PNGs del requisitado (incluye prototipos SALT)
images/RUP/01-analisis/...    ← SVGs de los diagramas de colaboración
images/RUP/02-diseño/...      ← SVGs de los diagramas de secuencia
```

Cada `.svg` tiene su fuente `.puml` en la ruta análoga bajo [`/modelosUML/`](/modelosUML/). Las READMEs de cada artefacto bajo `RUP/` enlazan a ambos.
